Directions

Use the arrow keys to steer the ship. Avoid sea monsters and collect coins. If you run into a sea monster, game over.

Description

This game uses sprites to load in images for the characters from the p5.js library. If you collide with a sea monster, the screen is set to a loose screen, but you can reset it by clicking 'r' or 'p' . This is done with the keyDown function. It is also the windowWidth function to make it a full screen game.
