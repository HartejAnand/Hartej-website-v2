Description

This program uses the p5.js library to create a sprite with a random velocity. If the sprite hits the edges of the screen, the velocity is reversed by multiplying it by -1. The program uses a function to transfer the color of the border boxes to the bouncing square on collision. If the bouncing square hits the black box in the center, it will freeze and play a sound using the NAME.play(); function that is included in the p5.js library.
