Directions

You click the 'moles' with the red box. For every mole you hit, you get a point. Be careful, they are fast.

Description

This game uses simple graphics [squares]. The moles randomly appear, and if they are clicked while visible=true, the point variable increasses. There is no time limit, so you can get as many points as you want.
